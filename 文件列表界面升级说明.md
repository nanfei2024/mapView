## 💡 使用技巧

### 搜索文件
1. 在搜索框输入文件名
2. 自动实时过滤
3. 或使用类型筛选器

### 排序列表
1. 点击列标题（文件名称、修改时间、类型）
2. 观察排序图标变化
3. 再次点击切换方向

### 批量删除
1. 勾选要删除的文件
2. 点击"批量删除"按钮
3. 确认删除操作

### 快速跳转
1. 点击页码按钮
2. 或在"Go to"框输入页码
3. 按 Enter 跳转

## 📊 性能优化

### 渲染优化
```typescript
// 使用计算属性缓存
const visiblePages = computed(() => {
  // 智能计算可见页码
  // 避免渲染过多页码按钮
});

// 条件渲染
v-if="files.length === 0"  // 空状态
v-for="file in files"       // 文件列表
```

### 样式优化
```css
/* 使用 transform 而非 position */
transform: translateY(-2px);  /* ✅ GPU 加速 */
top: -2px;                    /* ❌ 触发重排 */

/* 使用 CSS 变量（可选） */
--primary-color: #667eea;
--border-radius-lg: 12px;
```
