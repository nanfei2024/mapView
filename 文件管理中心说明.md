# 📋 文件管理中心说明

## ✨ 功能概述

文件管理中心是一个**三合一**的综合管理页面，集成了：
1. 📋 **文件列表** - 管理各类文件
2. 🖼️ **我的图库** - 管理图片资源
3. 📊 **我的表库** - 管理数据表格

通过标签页切换，一个页面搞定所有文件资源管理！

---

## 🎯 页面结构

```
┌─────────────────────────────────────────────────────┐
│ ← 返回  文件管理中心                       👤 管理员│
├─────────────────────────────────────────────────────┤
│ 📋 文件列表  |  🖼️ 我的图库  |  📊 我的表库        │ ← 标签切换
├─────────────────────────────────────────────────────┤
│                                                      │
│                    内容区域                          │
│                  (根据选择显示)                       │
│                                                      │
└─────────────────────────────────────────────────────┘
```

## 🚀 使用方法

### 访问页面

**从主页进入：**
1. 点击主页右侧控制面板的 **"📋 文件管理"** 按钮
2. 自动跳转到文件管理中心
3. 默认显示"文件列表"标签

**直接访问：**
```
http://localhost:5173/file-list
```

### 切换标签

点击顶部标签按钮即可切换：
- 📋 **文件列表** - 查看和管理所有文件
- 🖼️ **我的图库** - 查看和管理图片
- 📊 **我的表库** - 查看和管理数据表格

---

## 📋 标签一：文件列表

### 功能特点
- ✅ 文件列表展示
- ✅ 搜索文件
- ✅ 类型筛选
- ✅ 排序功能
- ✅ 批量删除
- ✅ 详情查看/编辑/删除

### 操作说明
1. **搜索**：在搜索框输入文件名
2. **筛选**：选择文件类型进行筛选
3. **排序**：点击列标题进行排序
4. **批量操作**：勾选文件后点击"批量删除"

---

## 🖼️ 标签二：我的图库

### 功能特点
- ✅ **双视图模式**：网格视图 / 列表视图
- ✅ **上传图片**：支持图片上传
- ✅ **分类管理**：地貌/地质/矿物/化石等
- ✅ **图片预览**：点击查看大图
- ✅ **批量操作**：批量删除图片

### 视图切换
```
⊞ 网格视图 - 以卡片形式展示图片
☰ 列表视图 - 以表格形式展示详细信息
```

### 网格视图特点
- 图片卡片展示
- 悬停显示操作按钮
- 显示分类、大小、日期
- 快速预览和下载

### 列表视图特点
- 表格详细信息
- 缩略图预览
- 完整元数据展示
- 批量操作支持

### 操作说明
1. **上传图片**：点击"📤 上传图片"按钮
2. **查看图片**：
   - 网格视图：点击图片或悬停点击"👁️ 预览"
   - 列表视图：点击缩略图或操作栏"👁️"
3. **下载图片**：点击"⬇️ 下载"按钮
4. **编辑/删除**：点击相应的操作按钮

---

## 📊 标签三：我的表库

### 功能特点
- ✅ **表格卡片展示**：直观的卡片式布局
- ✅ **多格式支持**：Excel / CSV / JSON
- ✅ **分类管理**：地质数据/矿物成分/地貌特征等
- ✅ **详细统计**：显示行数、列数、文件大小
- ✅ **导入/新建**：支持导入和新建表格

### 卡片信息
每个表格卡片显示：
```
┌────────────────────────────────┐
│ ☑️  📊  👁️ ⬇️                    │
│                                │
│ 表格名称                        │
│ 描述信息...                     │
│                                │
│ 📝 156行  📋 12列  💾 458KB    │
│                                │
│ Excel   地质数据    ✏️ 🗑️      │
│ 更新于 2025-11-09               │
└────────────────────────────────┘
```

### 操作说明
1. **导入表格**：点击"📥 导入表格"按钮
2. **新建表格**：点击"➕ 新建表格"按钮
3. **查看详情**：点击卡片上的"👁️ 查看"按钮
4. **下载表格**：点击"⬇️ 下载"按钮
5. **编辑/删除**：点击底部的相应按钮

## 🔧 技术实现

### 文件结构
```
src/
├── views/
│   └── FileListPage.vue        # 主页面（包含三个标签）
└── components/
    ├── FilePagination.vue      # 文件列表组件
    ├── ImageGallery.vue        # 图库组件
    └── TableLibrary.vue        # 表库组件
```

### 标签切换实现
```typescript
// 当前激活的标签
const activeTab = ref<'files' | 'images' | 'tables'>('files');

// 模板中使用
<div v-if="activeTab === 'files'">
  <FilePagination />
</div>
<div v-if="activeTab === 'images'">
  <ImageGallery />
</div>
<div v-if="activeTab === 'tables'">
  <TableLibrary />
</div>
```

### 样式特点
```css
/* 标签容器固定在顶部 */
.tabs-container {
  position: sticky;
  top: 86px;
  z-index: 999;
}

/* 标签按钮 */
.tab-btn {
  border-bottom: 3px solid transparent;
  transition: all 0.3s ease;
}

.tab-btn.active {
  color: #667eea;
  border-bottom-color: #667eea;
}
```

## 🆘 常见问题

### Q: 如何切换标签？
A: 点击页面顶部的标签按钮即可切换。

### Q: 数据会保存吗？
A: 当前使用模拟数据，刷新后重置。后端对接后会持久化。

### Q: 支持哪些文件格式？
A: 文件列表支持所有格式，图库支持图片格式，表库支持Excel/CSV/JSON。

### Q: 如何批量删除？
A: 勾选要删除的项目，然后点击"批量删除"按钮。

### Q: 可以同时管理多个账户吗？
A: 当前为单用户版本，多用户功能将在后续版本中支持。
