# 🚀 快速体验文件列表页面

## ⚡ 一键启动

### Windows 用户
双击运行 `启动服务.bat`

### 或使用命令行
```bash
cd mapView-main
npm run dev
```

---

## 📋 体验步骤

### 方法一：从主页跳转（推荐）

1. **打开浏览器** 访问 `http://localhost:5173/`

2. **找到文件列表按钮**
   - 在主页右侧控制面板
   - 找到 📋 **文件列表** 按钮

3. **点击跳转**
   - 点击按钮
   - 自动跳转到文件列表页面

4. **返回主页**
   - 点击左上角 **← 返回** 按钮

### 方法二：直接访问

浏览器地址栏输入：
```
http://localhost:5173/file-list
```

## 💡 使用技巧

### 快速返回主页
1. 点击左上角 **← 返回**
2. 或按浏览器后退按钮
3. 或在地址栏输入 `/`

### 搜索文件
1. 在搜索框输入文件名
2. 实时过滤结果
3. 支持中英文搜索

### 筛选文件
1. 点击 **文件类型** 下拉框
2. 选择 PDF / Markdown / 图片
3. 点击 **🔄 重置** 清除筛选

### 批量删除
1. 勾选要删除的文件
2. 点击 **🗑️ 批量删除**
3. 确认删除操作

## 🔧 故障排除

### 页面无法访问
```bash
# 确保服务已启动
npm run dev

# 检查端口占用
netstat -ano | findstr :5173

# 清除缓存
Ctrl + Shift + R (Chrome/Edge)
```

### 样式显示异常
```bash
# 清除浏览器缓存
Ctrl + Shift + Delete

# 重新启动服务
Ctrl + C (停止)
npm run dev (重启)
```

### 路由跳转失败
```bash
# 检查路由配置
cat src/router/index.ts

# 确认组件已导入
cat src/views/FileListPage.vue
```
